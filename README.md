# Jaalee to Home Assistant via Shelly BLU Gateway & MQTT

[![Home Assistant](https://img.shields.io/badge/Home%20Assistant-Compatible-41BDF5.svg)](https://www.home-assistant.io/)

Parse Jaalee JHT BLE sensor data using Shelly BLU Gateway devices as Bluetooth proxies and forward to Home Assistant via MQTT Auto-Discovery.
